Лендинги для разных операторов
BY LifeMegafoneBeeline

У каждого оператора были чёткие правила по расположению и наполнению блоков с правовой информацией, а так же ограничения по общему объему контента, размерам блоков и контрастности кнопок

По требованиям операторов в лендинги не редко приходилось вносить правки. Довольно быстро я создала систему для оперативных внесений изменений во все лендинги (пара минут) и их создания с нуля (10-20 минут на один лединг)

Двухфакторные лендинги

Лендинги с участием JavaScript для MTC в Белоруссии и примерно такие же для Таджикистана

JS здесь не только делает кнопку не доступной, если в согласии не стоит галочка, но и меняет главную надпись

Лендинги для разных компаний
